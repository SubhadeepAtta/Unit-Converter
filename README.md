# Unit-Converter

A simple and responsive **Metric/Imperial Unit Converter** built using **HTML, CSS, and JavaScript**. The application converts between metric and imperial units for **length**, **volume**, and **mass**.

## Features

- Convert **Meters ↔ Feet**
- Convert **Liters ↔ Gallons**
- Convert **Kilograms ↔ Pounds**
- Clean and minimal UI
- Built with Vanilla JavaScript
- Responsive design

##  Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

##  Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/unit-converter.git
```

2. Navigate to the project folder

```bash
cd unit-converter
```

3. Open `index.html` in your browser.

No installation or dependencies are required.
